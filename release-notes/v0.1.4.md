## Eucalypso v0.1.4

This release improves usability and setup clarity.

### What's new
- Added built-in module help via `help.json`, based on the current README.
- Added clock status warnings when `Sync` is set to `clock`:
  - `Enable MIDI Clock Out in Move settings`
  - `Clock out enabled, transport stopped`
- Included `help.json` in packaged module builds (`dist/eucalypso/help.json`).

### Why this matters
- Easier parameter guidance directly in Move Everything.
- Faster troubleshooting when external clock sync is enabled but not running.

### Upgrade
- Install as usual from this release artifact and restart Move Anything.
