## Eucalypso v0.1.5

This release adds a new drum-focused register workflow.

### What's new
- Added `register_mode=drumpad` (fixed pad notes `36-51`).
- In `drumpad` mode, lane playback is gated by held/latching pads:
  - pad 1 gates lane 1
  - pad 2 gates lane 2
  - pad 3 gates lane 3
  - pad 4 gates lane 4
- Lane `Note` still chooses which pad note is emitted.
- Register range still applies in `drumpad` mode via `scale_rng` (uses `1-16` pads).

### Not changed
- `held` and `scale` register modes behave as before.

### Upgrade
- Install this release artifact and restart Move Anything.
